# BETA_RELEASE

## Scope

This beta validates domain adaptation from ecosystem templates to agriculture-specific federated learning workflows.

## Included

- Static geospatial dashboard simulation.
- Policy-gated federated learning workflow placeholders.
- Initial agriculture compliance evidence mapping.
- Baseline operator and governance documentation.

## Exit Criteria for GA Candidate

- Live telemetry integration for FL and compliance metrics.
- Region-policy enforcement tests (cross-border model flow constraints).
- Security wrapper integration tests for signed update verification.
- Reproducible artifact capture and CI quality gates.
