# COMPLIANCE_EVIDENCE

This document maps Sovereign Mohawk Agriculture Global controls to agriculture-relevant governance and privacy requirements.

## Control Mapping

| Platform Control | Example Regulatory Alignment | Evidence Pattern |
| --- | --- | --- |
| Data stays local to farm node by default | Farmer-owned data principles, GDPR-style minimization | Node policy config + update logs |
| Signed policy-gated model exchange | Cross-border data transfer constraints | Round policy signatures + audit trail |
| TPM attestation before trust elevation | Equipment integrity and device trust controls | Attestation records and quote verification logs |
| Differential privacy and secure aggregation | Confidentiality for farm-level operational telemetry | Aggregation policy and DP parameter snapshots |
| Regional deny/allow policies | EU CAP-aligned governance and regional sovereignty controls | Policy engine evaluation report |

## Operational Evidence to Retain

1. FL round policy signatures.
2. Regional transfer allow/deny decisions.
3. TPM attestation verification events.
4. Compliance scorecard snapshots.
5. Incident and exception records.
